━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
URL: https://fio42.github.io/safemergepdf/split-pdf/
canonical: /split-pdf/
title: Split PDF Free Online — No Upload, 100% Private | SafeMergePDF
Keywords: split pdf free, split pdf online, separate pdf pages,
  extract pages from pdf, divide pdf, pdf splitter no upload

Limites: 1 archivo, hasta 100 MB
Modos:
  "all"   — cada pagina se convierte en un PDF separado
  "range" — extrae un rango especifico (ej: 1-3, 5, 7-9) en un solo PDF

Python usado:
  MODO ALL:
    reader = PdfReader(BytesIO(bytes(file_bytes_js)))
    for idx in page_indices_js:
        writer = PdfWriter()
        writer.add_page(reader.pages[int(idx)])
        out = BytesIO(); writer.write(out)
        results.append(out.getvalue())

  MODO RANGE:
    writer = PdfWriter()
    for idx in page_indices_js:
        writer.add_page(reader.pages[int(idx)])
    writer.write(out)

JS globals: lang, selectedFile, pyodide, pyodideReady,
  splitMode, totalPages, objectURLs[]

Funciones JS:
  toggleLang, triggerFileInput, handleDragOver/Leave/Drop,
  handleFiles, removeFile, formatSize,
  setMode(mode): alterna entre 'all' y 'range', muestra/oculta range input
  parseRange(rangeStr, total): parsea "1-3,5,7-9" -> [0,1,2,4,6,7,8] (0-indexed)
    retorna null si el rango es invalido
  initPyodide, startSplit, setProgress, showResult,
  resetUI, resetFull, showError, esc

Flujo:
  1. Usuario sube PDF
  2. Drop zone se oculta, aparece file-preview + split-options
  3. Usuario elige modo (all o range)
  4. Si range: muestra input de texto para rango
  5. Click Split -> espera Pyodide -> lee bytes -> obtiene total paginas
  6. Modo all: un PdfWriter por pagina -> array de results
     Modo range: parsea rango -> un PdfWriter con paginas seleccionadas
  7. Genera Blob URLs -> muestra lista de links de descarga en result-area
  8. Cleanup Python globals

Resultado:
  Modo all: lista de links descargables "Page X — Y KB"
  Modo range: un link "Download extracted pages — Y KB"
  Nombre de archivos: {nombre-original}-page-{N}.pdf / {nombre}-pages-{rango}.pdf

MERGE:  PdfWriter().append(BytesIO(bytes)) x N -> write(output)
COMPRESS: compress_identical_objects(True,True) + img.replace(quality=N)
SPLIT ALL: por pagina: PdfWriter().add_page(page).write(out) -> array
SPLIT RANGE: PdfWriter() con paginas seleccionadas -> write(out)
ROTATE: page.rotate(90|180|270)
REORDER: writer.add_page(reader.pages[i]) en nuevo orden
DELETE: if i not in to_delete: writer.add_page(page)
PROTECT: writer.encrypt(password, algorithm="AES-256")
UNLOCK: reader.decrypt(password); writer.append(reader)
WATERMARK: fpdf2 para crear pagina watermark + merge con cada pagina
PDF TO TEXT: page.extract_text() -> Blob text/plain
