FAQ

Is my file uploaded to a server?

No. Your PDF file is never uploaded. Only the text extracted from it is sent to the translation API. The extraction itself runs 100% locally in your browser.

What is MyMemory?

MyMemory is a free public translation API that supports 100+ language pairs. It does not require an account or API key. It processes text, not files.

Can I translate confidential documents?

We advise caution. While your file stays local, the text content is sent to an external API for translation. For highly sensitive documents, use a private translation service.

Is there a page or size limit?

Files up to 20 MB. Very long documents may be split into chunks for translation. The free API has rate limits — large documents may take a moment.

Can I download the translation?

Yes — download as TXT (original or translated) or as a simple PDF. All generated locally in your browser.
