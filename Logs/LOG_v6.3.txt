CDN PDF.js:  https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js
             worker: https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js

MERGE:    PdfWriter.append(BytesIO) x N archivos -> merged.pdf
COMPRESS: compress_identical_objects + img.replace(quality=N) -> compressed.pdf
SPLIT:    writer.add_page por rango -> split.pdf
PDF->JPG: PDF.js page.render -> canvas -> toDataURL jpeg -> page-N.jpg
JPG->PDF: Pillow Image.save(format='PDF') -> filename.pdf
ROTATE:   page.rotate(90|180|270) scope all/range -> rotated.pdf
PROTECT:  writer.encrypt(user_password, use_128bit=True) -> protected.pdf
UNLOCK:   reader.decrypt(pw) casos: ok/wrong_password/not_encrypted -> unlocked.pdf

EDIT PDF (v6.3 — PDF.js + canvas + custom PDF builder):
  Stack: PDF.js 3.11.174 (render) + canvas API (anotaciones) + builder PDF manual
  NO usa Pyodide — 100% JavaScript nativo
  
  Modos:
    select   — mover/seleccionar elementos
    text     — <textarea> overlay sobre canvas, posicionado con click
    draw     — freehand con canvas strokeStyle, guardado como path {x,y}[]
    highlight— igual que draw pero globalAlpha=0.35, lineWidth grueso
    image    — drag+resize overlay con <img> sobre canvas
    sign     — modal con sigCanvas, draw firma, place como imagen PNG
  
  Estado interno:
    annotations[pageNum] = [{type:'draw'|'highlight', data:{path,color,width}}]
    textInputs[pageNum]  = [DOMElement textarea]
    imageOverlays[pageNum] = [DOMElement div con img]
  
  Descarga (buildPDFFromJpegs):
    Por cada pagina: renderizar PDF a offscreen canvas scale=2.0
    Redibujar anotaciones draw/highlight encima
    Redibujar texto de los textarea (fillText)
    Redibujar imagenes (drawImage)
    Exportar cada pagina como JPEG base64
    Ensamblar PDF manual con XObjects JPEG (PDF 1.4 spec)
    Blob -> URL.createObjectURL -> descarga filename-edited.pdf
  
  IMPORTANTE: el editor usa layout de pantalla completa (height:calc(100vh-54px))
  No usa Tailwind CDN (para no bloquear CSP). Usa CSS variables nativas.
