EDIT-PDF — ARQUITECTURA DEL TEXTO (v6.6)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Estructura DOM de cada texto:
  <div class="text-wrap" style="left:Xpx;top:Ypx">   <-- draggable wrapper
    <div class="text-drag-handle">...icono...Text [x]</div>  <-- drag aqui
    <textarea class="text-input-overlay">...</textarea>
  </div>

Comportamiento por modo:
  select:   annCanvas.pointerEvents='none' -> clicks pasan a text-wraps abajo
            arrastrar por el handle azul o por cualquier parte del wrap excepto textarea
  text:     click en canvas crea nuevo text-wrap en esa posicion
  draw/highlight: annCanvas captura eventos normalmente

Download (buildPDFFromJpegs):
  Itera textInputs[n] (array de text-wrap divs)
  Obtiene textarea dentro con wrap.querySelector('textarea')
  Usa wrap.style.left/top + 20px offset para saltar el handle bar
  NOTA: el +20px compensa la altura del handle en pantalla

NUNCA cambiar la estructura wrap/handle/textarea sin actualizar downloadPDF().

CDN PDF.js:  https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js
             worker: https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js

MERGE:    PdfWriter.append(BytesIO) x N archivos -> merged.pdf
COMPRESS: compress_identical_objects + img.replace(quality=N) -> compressed.pdf
SPLIT:    writer.add_page por rango -> split.pdf
PDF->JPG: PDF.js page.render -> canvas -> toDataURL jpeg -> page-N.jpg
JPG->PDF: Pillow Image.save(format='PDF') -> filename.pdf
ROTATE:   page.rotate(90|180|270) scope all/range -> rotated.pdf
PROTECT:  writer.clone_reader_document_root + writer.encrypt(user_password, use_128bit=True)
UNLOCK:   reader.decrypt(pw) casos: ok/wrong_password/not_encrypted -> unlocked.pdf
EDIT PDF: PDF.js (render) + canvas API (draw/highlight) + text-wrap divs (texto)
          + img-overlay divs (imagenes/firma) + buildPDFFromJpegs (descarga)
