Stack base: Pyodide v0.25.0 + pypdf (via micropip)
CDN Pyodide: https://cdn.jsdelivr.net/pyodide/v0.25.0/full/pyodide.js
CDN PDF.js: https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js

MERGE PDF (Pyodide + pypdf):
  PdfWriter().append(BytesIO(bytes)) por cada archivo
  writer.write(output_buffer) -> descarga merged.pdf

COMPRESS PDF (Pyodide + pypdf):
  compress_identical_objects(remove_identicals=True, remove_orphans=True)
  Por imagen: img.replace(img.image, quality=N) segun nivel elegido
  Descarga filename-compressed.pdf

SPLIT PDF (Pyodide + pypdf):
  PdfWriter().add_page(page) por paginas del rango
  Descarga filename-split.pdf

PDF -> JPG (PDF.js, NO Pyodide):
  page.render() -> canvas -> canvas.toDataURL('image/jpeg')
  Descarga page-N.jpg

JPG -> PDF (Pyodide + Pillow):
  Image.open(BytesIO(bytes)) -> img.save(buf, format='PDF')
  PdfWriter().append() -> descarga filename.pdf

ROTATE PDF (Pyodide + pypdf):
  page.rotate(angle) donde angle es 90, 180 o 270
  Scope: all pages o rango especifico (parse_range)
  Descarga filename-rotated.pdf

PROTECT PDF (Pyodide + pypdf):
  writer.clone_reader_document_root(reader)
  writer.encrypt(user_password=pw, owner_password=None, use_128bit=True)
  Descarga filename-protected.pdf
  UI: password input + confirm + strength indicator + show/hide toggle

UNLOCK PDF (Pyodide + pypdf):
  reader.is_encrypted -> reader.decrypt(password) -> retorna 0=fallo, 1=exito
  writer.clone_reader_document_root(reader) -> writer.write(out)
  Maneja caso: PDF no estaba encriptado (lo descarga igual)
  Maneja caso: password incorrecta (muestra error claro)
  Descarga filename-unlocked.pdf
